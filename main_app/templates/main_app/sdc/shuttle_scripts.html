Shuttle scripts

Files can be edited before sending. To use this logic, simply place the necessary executable file (in the following script) in a folder reserved for this purpose.

  • Linux: ~/shuttle/scripts
  • Windows: %USERPROFILE%\shuttle\scripts

This folder can be created manually or is automatically generated when the shuttle is started for the first time.

All executable files within this folder (Windows: .exe; Linux: any executable file) are executed in alphabetical order.

When a script is executed, it receives the path of the file to be sent as an argument. After all scripts have completed, all files from the folder where the original file was located are uploaded to the server. This means that the original file can be edited and overwritten, or new files can be created in the same folder.

File from content renamer

This Script reads a file and renames it based on its content. To use it you need to add a .env file into the script directory with the following keys.

The .env file

SFC_FILE_FILTER_RE=\.txt$
SFC_NEW_NAME_RE=^Data File Name\s+(.+)$

SFC_FILE_FILTER_RE is used as a filter. If this regexp matches the file will be renamend

SFC_NEW_NAME_RE is used to find the new name. This regex must have at least one group. The new name is the match of the first group

ShuttleFromContentRenamer (Linux) or ShuttleFromContentRenamer (Win)

ChemConverter

ChemConverter is a file-format conversion service used in conjunction with the Chemotion Electronic Lab Notebook (ELN). Its main purpose is to standardize analytical data files — converting them into formats that Chemotion (and related tools like ChemSpectra) can display and work with.

How It Works Technically

ChemConverter has a three-layer architecture:
Reader layer:
Code that reads the incoming file format (TXT, CSV, DTA, etc.) and parses columns and values.

Admin profile layer:
Rules defined (by admins) for how to recognize and transform different input formats — based on file extension, header lines, column titles, or other identifiers.

Processing & Output:
The converter applies the selected profile, outputs standardized formats, and stores metadata and data in structures Chemotion understands.
